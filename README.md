# Artist Portfolio – Full Stack Project

A full-stack artist portfolio website built with a custom backend, database integration, and an admin dashboard to manage enquiries.

This project demonstrates real-world full-stack development skills including frontend design, backend APIs, authentication, and deployment.

---

## Live Demo
- Frontend: https://artist-portfolio-frontend.netlify.app/
- Backend API: https://artist-portfolio-backend-ht7x.onrender.com/

---

## Features
- Responsive multi-page artist portfolio
- Enquiry form connected to backend API
- PostgreSQL database (Supabase)
- Admin login with JWT authentication
- Admin dashboard to view enquiries
- Mark enquiries as “Contacted”
- Deployed frontend and backend

---

## Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- Node.js
- Express
- PostgreSQL (Supabase)
- JWT Authentication

**Deployment**
- Netlify (Frontend)
- Render (Backend)
